## üöÄ Getting Started

### What is Kashis Studio?

Kashis Studio is a modern WordPress theme designed specifically for studio rental businesses. It provides:

- **Beautiful Design**: Professional look based on Atlassian Design System
- **Easy to Use**: No coding required for most tasks
- **Custom Blocks**: 6 specialized blocks for studio rental content
- **Mobile Friendly**: Looks great on all devices
- **Fast Performance**: Optimized for speed

### System Requirements

- **WordPress**: Version 6.4 or higher
- **PHP**: Version 7.4 or higher
- **Browser**: Modern browser (Chrome, Firefox, Safari, Edge)

## üé® Theme Setup

### First-Time Activation

When you activate Kashis Studio for the first time, you'll see a **Welcome Screen** with setup options:

1. **Generate Sample Content**: Click this to create example rooms and pages
2. **Open Customizer**: Jump directly to theme customization
3. **Visit Settings**: Configure theme options
4. **View Showcase**: See all available blocks in action

### Quick Setup Steps

1. **Activate the Theme**
   - Go to **Appearance ‚Üí Themes**
   - Find **Kashis Studio**
   - Click **Activate**

2. **Choose a Color Preset**
   - Go to **Appearance ‚Üí Customize**
   - Click **Kashis Studio Settings ‚Üí Colors**
   - Select a preset (Default, Warm, Cool, Professional, Elegant)
   - Click **Publish**

3. **Set Your Logo**
   - In Customizer, go to **Site Identity**
   - Upload your logo
   - Set site title and tagline
   - Click **Publish**

4. **Configure Menus**
   - Go to **Appearance ‚Üí Menus**
   - Create a new menu
   - Add pages (Home, Rooms, About, Contact)
   - Set as **Primary Menu**
   - Save

## üè¢ Managing Studio Rooms

### Creating a New Room

1. Go to **Studio Rooms ‚Üí Add New**
2. Enter room title (e.g., "Studio A - 25„é°")
3. Add description in the editor
4. Set a **Featured Image** (room photo)
5. Add **Excerpt** (short description)
6. Click **Publish**

### Room Information Tips

**Title**: Be descriptive
- ‚úÖ Good: "Studio A - Large Space with Natural Light"
- ‚ùå Bad: "Room 1"

**Description**: Include key details
- Size (square meters)
- Capacity (number of people)
- Equipment available
- Pricing information
- Booking instructions

**Featured Image**: Use high-quality photos
- Recommended size: 1200x800px
- Show the space clearly
- Good lighting
- Clean and organized

### Organizing Rooms by Category

1. Go to **Studio Rooms ‚Üí Room Categories**
2. Add new categories:
   - Small (under 20„é°)
   - Medium (20-40„é°)
   - Large (40„é°+)
   - Special Features (with piano, recording studio, etc.)
3. Assign categories when editing rooms

### Room Custom Fields (Optional)

You can add custom fields for additional information:

- **Size**: Room size in square meters
- **Capacity**: Maximum number of people
- **Hourly Rate**: Price per hour
- **Daily Rate**: Price per day
- **Equipment**: List of available equipment

## üß± Using Custom Blocks

Kashis Studio includes 6 custom blocks to showcase your studio rooms and information.

### 1. Room List Block

Displays a grid of your studio rooms.

**How to Use:**
1. Edit a page
2. Click **+** to add block
3. Search for "Room List"
4. Configure in sidebar:
   - **Posts Per Page**: How many rooms to show
   - **Columns**: 1-4 columns
   - **Show Filters**: Enable category filtering
   - **Order By**: Sort by date, title, or modified date

**Best For**: Main rooms listing page, homepage showcase

### 2. Pricing Table Block

Display pricing plans in a beautiful table.

**How to Use:**
1. Add the "Pricing Table" block
2. Edit the plans in the sidebar (JSON format)
3. Mark one plan as "featured" for highlighting
4. Customize button text and colors

**Example Plans:**
- Basic: ¬•5,000/hour
- Standard: ¬•8,000/hour
- Premium: ¬•12,000/hour

**Best For**: Pricing page, service comparison

### 3. Accordion/FAQ Block

Create collapsible FAQ sections.

**How to Use:**
1. Add the "Accordion" block
2. Add multiple items:
   - Question as title
   - Answer as content
3. Toggle "Open First" to expand first item by default

**Common FAQs:**
- How do I book a room?
- What equipment is included?
- Can I cancel or reschedule?
- Are there any additional fees?
- What are your operating hours?

**Best For**: FAQ page, booking information

### 4. Timeline Block

Show your studio's history or rental process.

**How to Use:**
1. Add the "Timeline" block
2. Add timeline items:
   - Date (e.g., "2020" or "Step 1")
   - Title
   - Description
3. Items appear in vertical timeline

**Use Cases:**
- Studio history milestones
- Booking process steps
- Event timeline
- Renovation history

**Best For**: About page, process explanation

### 5. Testimonial Block

Display customer reviews and ratings.

**How to Use:**
1. Add the "Testimonial" block
2. Fill in:
   - Customer name
   - Rating (1-5 stars)
   - Review text
   - Optional photo
3. Style appears as a card with quote styling

**Tips:**
- Ask satisfied customers for reviews
- Include photos with permission
- Keep reviews authentic
- Show variety of customers

**Best For**: Homepage, testimonials page

### 6. Call-to-Action Block

Create prominent call-to-action sections.

**How to Use:**
1. Add the "CTA" block
2. Configure:
   - Headline
   - Description
   - Button text
   - Button URL

3. Use for important actions

**Use Cases:**
- "Book Your Studio Now"
- "Request a Tour"
- "Download Price List"

**Best For**: Bottom of pages, conversion points

## üìù Using Shortcodes

Shortcodes let you insert complex content with simple text codes.

### [kashis_rooms] - Display Room Grid

**Basic Usage:**
```
[kashis_rooms]
```

**With Parameters:**
```
[kashis_rooms limit="9" columns="3" category="large"]
```

**Parameters:**
- `limit`: Number of rooms (default: 6)
- `columns`: Grid columns 1-4 (default: 3)
- `category`: Filter by category slug
- `style`: "grid" or "list" (default: grid)

**Examples:**

Show 12 rooms in 4 columns:
```
[kashis_rooms limit="12" columns="4"]
```

Show only large rooms:
```
[kashis_rooms category="large" limit="6"]
```

List view with 8 rooms:
```
[kashis_rooms style="list" limit="8"]
```

### [kashis_contact] - Contact Information

Displays your contact information in a formatted box.

**Usage:**
```
[kashis_contact]
```

Shows:
- Phone number
- Email address
- Physical address
- Business hours

**Where to Use**: Contact page, footer, sidebar

### [kashis_hours] - Business Hours

Displays your business hours in a table.

**Usage:**
```
[kashis_hours]
```

Shows operating hours for each day of the week.

**Where to Use**: Contact page, homepage, about page

### [kashis_access] - Location & Access

Displays location and directions.

**Usage:**
```
[kashis_access]
```

Shows:
- Address
- Nearest station
- Walking directions
- Parking information

**Where to Use**: Contact page, access/location page

## üîß Troubleshooting

### Common Issues

#### Theme Looks Broken After Activation

**Solution:**
1. Go to **Settings ‚Üí Permalinks**
2. Click **Save Changes** (no need to change anything)
3. Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)

#### Rooms Not Displaying

**Check:**
- Are rooms published? (Not drafts)
- Is there at least 1 room?
- Check block settings (posts per page, filters)

**Solution:**
1. Create at least one room
2. Ensure it's published
3. Refresh the page

#### Colors Not Changing

**Solution:**
1. In Customizer, click **Publish** (don't just save)
2. Clear browser cache
3. Check if using child theme override

#### Search Widget Not Showing

**Solution:**
1. Check widget area is visible in theme
2. Ensure at least one room exists
3. Verify widget is configured correctly

#### Images Not Displaying

**Solution:**
1. Re-upload images
2. Regenerate thumbnails: **Tools ‚Üí Regenerate Thumbnails**
3. Check file permissions on wp-content/uploads/

### Performance Issues

If your site is slow:

1. **Install a caching plugin**: WP Super Cache or W3 Total Cache
2. **Optimize images**: Use ShortPixel or Smush
3. **Use a CDN**: Cloudflare (free) or similar
4. **Limit plugins**: Deactivate unused plugins
5. **Choose good hosting**: Upgrade if on shared hosting

### Browser Compatibility

Theme is tested on:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

If you see issues on older browsers (IE11), consider displaying an upgrade message.

## ‚ùì FAQ

### General Questions

**Q: Do I need coding skills to use this theme?**
A: No! All features can be used through the WordPress admin interface.

**Q: Can I translate the theme?**
A: Yes, the theme is translation-ready. Use Loco Translate plugin or create .po/.mo files.

**Q: Is the theme mobile-friendly?**
A: Yes, fully responsive on all devices.

**Q: Can I use this theme for other businesses?**
A: While designed for studios, it can be adapted for other rental businesses.

### Customization Questions

**Q: How do I change fonts?**
A: Go to **Appearance ‚Üí Customize ‚Üí Kashis Studio Settings ‚Üí Typography**.

**Q: Can I add custom CSS?**
A: Yes, in **Appearance ‚Üí Customize ‚Üí Additional CSS**.

**Q: How do I create a child theme?**
A: See the Developer Guide in docs/DEVELOPER_GUIDE.md.

**Q: Can I modify the color presets?**
A: Yes, through Customizer or by editing CSS variables.

### Room Management Questions

**Q: How many rooms can I add?**
A: Unlimited! WordPress has no limit on posts.

**Q: Can I import rooms from a spreadsheet?**
A: Yes, use the WP All Import plugin with CSV file.

**Q: How do I feature certain rooms?**
A: Use the "Sticky" option in the room editor sidebar.

**Q: Can I add custom fields to rooms?**
A: Yes, use Advanced Custom Fields (ACF) plugin.

### Booking Questions

**Q: Does the theme include booking functionality?**
A: No, but you can integrate with booking plugins like:
- Booking Calendar
- WooCommerce Bookings
- Amelia
- SimplyBook.me

**Q: How do I add a booking button?**
A: Use the CTA block or add a button to rooms linking to booking page.

### Technical Questions

**Q: What's the parent theme?**
A: Twenty Twenty-Four (official WordPress theme).

**Q: Can I use page builders?**
A: Yes, but it's designed for the Block Editor (Gutenberg).

**Q: Is it compatible with WooCommerce?**
A: Yes, though not specifically styled for it.

**Q: Does it work with multisite?**
A: Yes, fully compatible with WordPress multisite.

### Support Questions

**Q: Where can I get help?**
A:
- Documentation: docs folder
- WordPress Forums: wordpress.org/support
- GitHub Issues: For bug reports

**Q: How do I report a bug?**
A: Create an issue on GitHub with:
- WordPress version
- Theme version
- Steps to reproduce
- Screenshots if applicable

**Q: Can I request new features?**
A: Yes! Submit feature requests on GitHub.
